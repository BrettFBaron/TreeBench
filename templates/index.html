README

This project examines how language models navigate multi-step decision-making scenarios, revealing potential biases and mode collapse tendencies.

The Experiment: We use a two-step decision tree where models are asked to:

  1. Choose a travel destination (France, Japan, Brazil, Australia, or Italy)
  2. Then select an activity at that destination (Museum, National Park, Beach, Restaurant, or Nightclub)

This structure allows us to observe:

  • Which paths through the decision tree models prefer
  • How consistently models choose the same options (mode collapse)
  • How the first decision influences the second (preference cascades)

This work builds on research by davidad (color benchmarking), canaletto (LessWrong analyses), Cleo Nardo (simulators paper), and gwern (mode collapse).
